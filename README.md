# Mohamed Gameel Elsayed — Portfolio

A bilingual (English/Arabic), responsive personal portfolio for Mohamed Gameel Elsayed, Flutter & Laravel developer and founder of Codiryx.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production files will be generated in `dist/` and can be deployed to GitHub Pages, Vercel, or Netlify.
